# ScoreScanner

Este repositorio contiene el código fuente para ScoreScanner, Una aplicación
que usa servicios OMR para digitalizar partituras impresas, llevándolas a un
formanto MusicXML.

## Información Adicional
ScoreScanner trabaja con:
* Python 3.7.3
* Audiveris
* Tesseract OCR
